## Troubleshooting

### Problem: "Still seeing old IP addresses"

**Solutions:**
1. Wait longer (DNS can take up to 48 hours)
2. Clear your DNS cache: `ipconfig /flushdns`
3. Try a different browser or incognito mode
4. Check with online tools (dnschecker.org)
5. Contact your domain registrar support

### Problem: "Can't find DNS records in my registrar"

**Solution:**
- Check if domain is registered at that provider
- Check if nameservers are pointed elsewhere
- Run `nslookup -type=NS farcastmints.com` to find nameservers

### Problem: "Changes not saving"

**Solution:**
- Make sure you clicked Save/Apply
- Try different browser
- Contact registrar support
- Check if account has admin permissions
